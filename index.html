Welcome to Movie Search

This project is a movie and TV show discovery website that uses the TMDB API to fetch and display entertainment content. Users can search for movies, TV series, and actors, view popular titles, and explore detailed information such as ratings, release dates, and descriptions. The interface focuses on a visual and user-friendly design with large posters, making it easy to browse and discover new content.
